# Get S3 object URL List

- 프로젝트 이미지의 저장소를 S3로 이전하고 이미지 url의 데이터 더미를 만들기 위해 작성
- 이미지(object)의 URL을 JSON파일로 만듬
- 아래 링크를 참고

# Usage

`npm i`<br><br>
`.env.sample`을 참고하여 `.env` 작성<br><br>
`node app.js`<br><br>
`_results` 폴더에 object url 리스트를 담은 json파일이 저장됨<br><br>
